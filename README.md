# NuxtFusion

## Env
```bash
# use Node.js v22 LTS
nvm use
```

## Setup
```bash
# Use npm or whatever you like to install the dependencies
npm install

# Development Server
npm run dev

# Build the application for production
npm run build

# Locally preview production build
npm run preview
```
